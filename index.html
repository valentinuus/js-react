10. Репозиторий GIT

git config --local user.name "Valchik" - установка пораметров репозитория git

git config --local user.email "valentinus@list.ru"

git config --global user.name "Valchik"

git config --global user.email "valentinus@list.ru"

3 состояния файлов, созданны, git следит(файлы попадают в индекс), git coplfk контрольную точку(комит)
