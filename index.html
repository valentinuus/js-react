10 Репозиторий GIT

В командной строке переход на другой диск - "D:", в другую папку на другом диске "cd /d d:\projects"

git init - инициализация проекта

git config --local user.name "Valchik" - установка пораметров репозитория git git config --local user.email "valentinus@list.ru" git config --global user.name "Valchik" git config --global user.email "valentinus@list.ru"

3 состояния файлов, созданны, git следит(файлы попадают в индекс), git coplfk контрольную точку(комит)

git status посмотреть статус репозитория git add -A - до бавляет все файлы в репозитории git add main.css git commit -a -m"lessons 10 GIT репозиторий" - создание коммита GIT test
